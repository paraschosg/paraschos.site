# paraschos.site

Personal portfolio site for George Paraschos — Information & Communication Systems Engineering student at the University of the Aegean.

Built with vanilla HTML, CSS, and JavaScript. Styled as a VS Code editor.

## Features

- VS Code-inspired UI with file tabs and sidebar
- Interactive terminal with custom commands
- Live GitHub activity feed
- GitHub contributions graph
- Blog system via `posts.json`
- Dark/Light mode
- Typewriter animations
- Ambient screensaver mode
- PWA — installable as an app
- Multiplayer presence counter

## Stack

- HTML / CSS / JavaScript (no frameworks)
- GitHub Pages (hosting)
- Cloudflare (DNS + email routing)
- Supabase (realtime presence)
- Google Analytics
